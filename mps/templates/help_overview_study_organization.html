• Study Organization

    In the {{ all_glossary.mpsdb_term }}, a {{ all_glossary.study_term }} contains all data and metadata associated to an experiment designed to assess {{ all_glossary.mpsmodel_term }} response. A list of studies in the {{ all_glossary.mpsdb_term }}, with summary information about the {{ all_glossary.mpsmodel_term }}s used in the {{ all_glossary.study_term }} and associated assay results, is provided on the Study List page. A {{ all_glossary.study_term }} can contain multiple {{ all_glossary.studytreatmentgroup_term }}s and a {{ all_glossary.studytreatmentgroup_term }} can include multiple {{ all_glossary.studyitem_term }}s. Each {{ all_glossary.studyitem_term }} can have multiple {{ all_glossary.assaydatapoint_term }}s, {{ all_glossary.assayimage_term }}s, and/or {{ all_glossary.assayvideo_term }}s associated to it.

    Component Description
    Study Treatment Group MPS models constructed and treated the same way are part of a study treatment group. The study treatment group is assigned a name during study setup. When an {{ all_glossary.mpsmodel }} is constructed using a chip device, multiple chips can be assigned to the same study treatment group. When an {{ all_glossary.mpsmodel }} is constructed using a plate device, wells from one or more plates can be assigned to the same study treatment group, or to different study treatment groups.
    Study Item An experimental {{ all_glossary.mpsmodel }} consisting of one chip, one well on a plate, multiple wells on a plate that function as one MPS model, multiple substructures within an individual well on a plate (e.g., skeletal myobundle), or some other custom configuration.
    Assay Data Points (non-omic data) Data points are assay targets measured at a specific time relative to the start of an experiment and from a specific location in an MPS model. Processed assay data can be directly uploaded to a study for immediate presentation on the Study Summary page via the MPS import friendly columnar (MIF-C) file format. Raw assay plate reader data, exported from an assay plate reader, can be uploaded to a study and processed using the plate reader data processing tool provided in the database. After processing, those data join with other processed data and are available for presentation on the Study Summary page. All processed assay data that have been submitted for presentation on the Study Summary page can be visualized/analyzed within the individual study (intra-study), across studies (inter-study), and/or in combination with curated reference data.
    Omic Data Points Due to the number of targets (i.e. gene references) associated with genomic data, these data (e.g., log 2 fold change and associated statistics, normalized counts) are uploaded to the MPS-Db via an interface specifically designed for genomic data.
    Assay Images & Videos Images and videos can be linked to a study item and displayed within a study. Currently, images and videos must be added to the MPS-Db by the database administrators. An image/video metadata file, along with the images/videos can be provided to the database administrators through box/drive sharing.
