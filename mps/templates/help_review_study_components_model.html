    There are several options for editing Matrix Items, including editing by group, this is just one option.

    • To edit a Matrix Item, click on the View/Edit button next to the Matrix Item of interest:
      • In the Matrix Item Details section
        • Add/Edit the information as appropriate (add additional details if desired)
        • If possible, add the MPS model protocol (if not added in the Matrix)
      • In the Compounds section
        • Edit the information as appropriate (add additional details if desired)
        • The addition time should be since the start (time 0) of the experiment
        • The duration is the length of time of exposure (if a bolus dose, use 1 minute; if the whole experiment, use the length of the experiment)
        • If you do not know supplier or lot, enter "unspecified" (suggestions are based on your previous entries)
        • If you do not know receipt date, leave it blank
      • In the Cell Samples section
        • Edit the information as appropriate (add additional details if desired)
        • When adding, click the magnifying glass to see a list of available cell samples
      • In the Settings section
        • Edit the information as appropriate (add additional details if desired)
      • After assay data have been added, the Exclude field of the tabular data can be edited (optional)
        • Review data in graphs and/or table
        • In the table, if desired, change the exclusion status of a data point (add an X to the Exclude field to omit by default, remove the X to include)
      • Click the Submit button to save changes

    • Help Notes:
      • Matrix Items, along with the Name/ID, Test Type, Setup Date, and Organ Model MUST be added through the Matrix, but can be changed on the Matrix Item Setup and Data page
      • When using scientific notation to enter values, use this format: 1e+3, 1e+10, 3e+15
      • The Exclude field, shown on the Matrix Item Setup and Data page is the only attribute of a data point that can be changed in the MPS-GUI
      • The individual Matrix Setup and Data page is the only place Matrix Item data are available in tabular format
      • For information on continuing to add MPS models and assay data to the Study (Step 4), click here
